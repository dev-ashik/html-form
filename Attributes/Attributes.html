autocomplete

<input type="text" placeholder="name" autocomplete="off" />

The input autocomplete attribute specifies whether a form or an input field should have autocomplete on or off.

Attributes

disabled

<input type="text" placeholder="disabled" disabled />

The value of a disabled input field will not be sent when submitting the form!

height and width

<input type="image" src="" alt="Submit" width="48" height="48" />

Tip: Always specify both the height and width attributes for images. If height and width are set, the space required for the image is reserved when the page is loaded. Without these attributes, the browser does not know the size of the image, and cannot reserve the appropriate space to it. The effect will be that the page layout will change during loading (while the images load).

maxLength

<input type="text" placeholder="maxLength='4'" maxLength="4" />

When a maxlength is set, the input field will not accept more than the specified number of characters. However, this attribute does not provide any feedback. So, if you want to alert the user, you must write JavaScript code.

min max

<input type="date" min="2022-08-01" max="2022-08-15" />

The input min and max attributes specify the minimum and maximum values for an input field. The min and max attributes work with the following input types: number, range, date, datetime-local, month, time and week.

multiple

<input type="file" multiple />

The input multiple attribute specifies that the user is allowed to enter more than one value in an input field.

pattern

<input type="text" placeholder="country_code" pattern="[A-Za-z]{3}" title="three letter country code" />

The input pattern attribute specifies a regular expression that the input field's value is checked against, when the form is submitted. The pattern attribute works with the following input types: text, date, search, url, tel, email, and password.

Tip: Use the global title attribute to describe the pattern to help the user.

placeholder

<input type="text" placeholder="ener you name" />

The short hint is displayed in the input field before the user enters a value.

readonly

<input type="text" value="readonly" readonly/>

The value of a read-only input field will be sent when submitting the form.

required

<input type="text" placeholder="username" required />

The input required attribute specifies that an input field must be filled out before submitting the form.

size

<input type="text" placeholder="disabled" size="500" />

The input size attribute specifies the visible width, in characters, of an input field. The default value for size is 20.

step

<input type="number" placeholder="age" step="3" />

The step attribute works with the following input types: number, range, date, datetime-local, month, time and week.

value

<input type="text" value="Bangladesh" />

The input value attribute specifies an initial value for an input field.
